rx.Propagator

Parallelizing

class Parallelizing extends Propagator[Future[Unit]]

A propagator which runs propagation waves on the given ExecutionContext. Exactly how it is done (e.g. serially or in parallel) depends on the ExecutionContext given. Returns a Future which will complete when the propagation wave is finished. This Future can be blocked on if you want to block until the propagation wave is complete before continuing.
